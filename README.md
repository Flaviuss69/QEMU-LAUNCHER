Add iso.sh in your folder with .iso or .qcow2 files or any other format.
Use chmod +x iso.sh
sudo bash iso.sh
it will automatically search for qemu-system-x86_64 on your system. 
I created this code to help people who compiled qemu from source to easily configure their VM instance. you can select ram and cpu count.
